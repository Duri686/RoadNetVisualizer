# RoadNet Visualizer

High-performance 3D multi-layer road‑network visualization and navigation simulator built with Three.js, Vite, and Web Workers. Features hierarchical pathfinding (cross‑floor stairs/elevators), obstacle‑aware routing, real‑time 3D interaction, and smooth path animation. See the language-specific README for details.

基于 Three.js、Vite 与 Web Workers 的高性能 3D 多层道路网络可视化与导航仿真平台，支持分层寻路（跨楼层楼梯/电梯）、避障路由、实时 3D 交互与平滑路径动画。详见对应语言的 README 文档。
